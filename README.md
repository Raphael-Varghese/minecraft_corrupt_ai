# How to use this pack
  ## Downloading and setting up
  ### The GUI way
  1. First, click `Code` at the top
  2. Next click `Download ZIP`
  3. Go to `File Explorer`, click `View`, then click `Show`, then `File Extensions`
  4. Extract the ZIP
  5. Compress the 2 folders inside(or go into `minecraft_corrupt_ai-main` then continue)
  6. rename both to `.mcpack` files (ex. `corrupt_AI_RP.zip` to `corrupt_AI_RP.mcpack`; and `corrupt_AI_BP.zip` to `corrupt_AI_BP.mcpack`)
  7. Delete this `README.md` file from the main folder.
  8. Delete the original `.zip` files and the original folders from the folder(there should only be 2 files: `corrupt_AI_BP.mcpack` and `corrupt_AI_RP.mcpack`)
  9. Go back and select the whole folder
  10. Right-click and select `Compress to...`
  11. Select `ZIP file`
  12. Accept and continue
  13. After it finishes, rename it from `minecraft_corrupt_ai-main.zip` to `corrupt_ai.mcaddon`
  14. Double-click to import into Minecraft
