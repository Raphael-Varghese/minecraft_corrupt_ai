# How to use this pack
  ## Downloading and setting up
  1. First, click `Code` at the top
  2. Next click `Download ZIP`
  3. Go to `File Explorer`, click `View`, then click `Show`, then `File Extensions`
  4. Extract the ZIP
  5. Delete this `README.md` file from there.
  6. Go back and select the whole folder
  7. Right-click and select `Compress to...`
  8. Select `ZIP file`
  9. Accept and continue
  10. After it finishes, rename it from `minecraft_corrupt_ai-main.zip` to `corrupt_ai.mcaddon`
  11. Double-click to import into Minecraft

**ONLY WORKS ON BEDROCK, NOT JAVA**
